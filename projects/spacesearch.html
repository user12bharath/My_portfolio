Space Search Engine

  • Tech Stack: Python, FastAPI, PostgreSQL, NASA API, LLM APIs
  • Role: Backend Developer
  • Github URL: Project Link

FastAPI backend for searching and analyzing space biology datasets using AI.

This search engine leverages NASA's extensive databases and uses Large Language Models (LLMs) to provide intelligent search capabilities for space biology research data.

Features include semantic search, data analysis, and integration with multiple NASA APIs for comprehensive space research data access.
